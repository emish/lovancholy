General Info

Introduction

Wise Words is a responsive HTML5 template with focus on typography and simplicity. It is primarily intended for text heavy applications, such as a blog. It is built on the Twitter Bootstrap framework, therfore many Bootstrap features and elements are available in Wise Words as well.

Browser Support

Wise Words supports IE8+, Firefox, Chrome, Safari, Opera and Android/iOS mobile Browsers.

Building an HTML Site with Wise Words

This section covers how to build a basic HTML page using Wise Words. This will enable you to create a basic page and fill it with your own content.

Basic HTML Structure

The basic HTML structure of Wise Words includes elements that are common to all pages such as CSS style sheets and JavaScript referneces. The body tag is empty except for one <div> element, which holds all content. Create a new file blank.html in the root HTML folder that looks like this:

<!DOCTYPE html>
<!--[if lt IE 7]>      <html class="no-js lt-ie9 lt-ie8 lt-ie7"> <![endif]-->
<!--[if IE 7]>         <html class="no-js lt-ie9 lt-ie8"> <![endif]-->
<!--[if IE 8]>         <html class="no-js lt-ie9"> <![endif]-->
<!--[if gt IE 8]><!--> <html class="no-js" lang="en"> <!--<![endif]-->
<head>
    <meta charset="UTF-8">
    <title>Wise Words - A typographic HTML template</title>

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <link href='http://fonts.googleapis.com/css?family=Poiret+One|PT+Serif|Open+Sans:400,300' 
          rel='stylesheet' type='text/css'>

    <link href="css/bootstrap.min.css" rel="stylesheet">
    <link href="css/bootstrap-responsive.min.css" rel="stylesheet">
    <link href="css/socialicons.css" rel="stylesheet">
    <link href="css/glyphicons.css" rel="stylesheet">
    <link href="css/halflings.css" rel="stylesheet">
    <link href="css/template.css" rel="stylesheet">
    <link href="css/colors/color-classic.css" rel="stylesheet" id="colorcss">

    <script src="js/modernizr.js"></script>

    <!-- HTML5 shim, for IE6-8 support of HTML5 elements -->
    <!--[if lt IE 9]>
      <script src="js/html5shiv.js"></script>
    <![endif]-->

</head>
<body>

<div class="container">

<!-- Content goes here! -->

</div>  <!-- End Container -->

<!-- Le javascript
================================================== -->
<!-- Placed at the end of the document so the pages load faster -->
<script src="js/jquery-1.9.1.js"></script>
<script src="js/bootstrap.js"></script>
<script src="js/tinynav.js"></script>
<script src="js/template.js"></script>

</body>
</html>

The Header: Logo and Navigation

Now you are ready to insert the header part, which contains the logo and the navigation. To do this, insert the following markup after the tag <div class="container">.

<div class="masthead clearfix">

<!-- The Logo -->
<a href="index.html"><img id="logo" src="img/wwlogo@2x.png" height="63" width="324" alt="Wise Words"></a>

<!-- The Navigation -->
<ul id="nav" class="nav ww-nav pull-right hidden-phone">
  <li><a href="#">Item 1</a></li>
  <li class="dropdown"><a href="#" class="dropdown-toggle" data-toggle="dropdown">Dropdown</a>
    <ul class="dropdown-menu">
        <li><a href="#">Item 2</a></li>
        <li><a href="#">Item 3</a></li>
    </ul>                   
  </li>
  <li><a href="#">Item 4</a></li>
</ul>
</div>

<hr>

Replace the logo with your own logo file if you wish and add/remove the navigation to your liking. The example above features two standard items as well as a dropdown menu with two child items.

Page Layouts and the Boostrap Grid System

Wise Words is based on Twitter Bootstrap. This means that the layout is built using the Bootstrap Scaffolding classes as described in the Boostrap Documentation. To gain a better understanding on how this works, read the section about the grid system.

For this example, we are going to include the simplest page layout of Wise Words, which is a single centered column of 6 column width, with an offset of 3.

Add the following markup right after the markup, which you added in the previous step.

<div class="row main-content">
    <div class="offset3 span6 zone-content">
        <article>
            <h1>Page Title</h1>
            <div>
                <p class="lead">
                    This is a lead sentence. 
                </p>
                <p>
                    This is a simple paragraph. Lorem ipsum dolor sit amet,
                    consectetur adipisicing elit. Rerum quas ipsa animi laborum soluta
                    excepturi minus omnis atque doloremque voluptatum assumenda aut 
                    beatae hic deleniti nulla. Natus eum officia ipsa!
                 </p>
            </div>
        </article>
    </div>
</div>

For other page layouts, see the various sample files which come with the template. You can adjust the layouts quite easily to your liking, using the Bootstrap classes. For example, try replacing the class offset3 with offset2, as well as span6 with span8. This will make the main content wider, which could be handy depending on the type of content you wish to publish.

<div class="offset2 span8 zone-content">

Footer Widgets and Copyright Line

If you want, you can add the three footer Widgets, the same way they are used in the examples. Add the following code below the snippet that you added in the previous step. Note that you are still inside the main content div <div class="container">.

<hr>
<div class="row triple-row">
    <div id="triple-first" class="span4 triple">
        <div class="zone zone-triple-first">
            <article class="widget-triple-first widget-html-widget widget">
                <header>
                    <h1>First Triple</h1>
                </header>
                <p>First Triple Content</p>
            </article>
        </div>
    </div>
    <div id="triple-second" class="span4 triple">
        <div class="zone zone-triple-second">
            <article class="widget-triple-second widget-html-widget widget">
                <header>
                    <h1>Second Triple</h1>
                </header>
                <p>Second Triple Content</p>
            </article>
        </div>
    </div>
    <div id="triple-third" class="span4 triple">
        <div class="zone zone-triple-third">
            <article class="widget-triple-third widget-html-widget widget">
                <header>
                    <h1>Third Triple</h1>
                </header>
                <p>Third Triple Content</p>
            </article>
        </div>
    </div>
</div>

Note that just like the page content div in the previous step, the triples are using the Bootstrap Grid classes (here span4) to determine their size. This makes it fairly easy to you to create your own combination of widgets, by using the various span and offset classes Bootstrap provides.

To add a copyright line on the very bottom, simply add 

<hr>
<div class="row">
    <div class="copyright span12">Copyright 2013</div>  
</div>

before the end of the container div.

Color Schemes

Customizing the Color Scheme

Wise Words includes multiple predefined color schemes. The colors of the template are all defined in a separate CSS file, so they can be easily customized and swapped.

The color style sheet is included in the <head> tag with the following line:

<link href="css/colors/color-classic.css" rel="stylesheet" id="colorcss">

You can find the color scheme css files in the subfolder HTML/css/colors. To change the scheme, simply include a different color CSS file into your site. For example:

<link href="css/colors/color-nytimes.css" rel="stylesheet" id="colorcss">

You can also create your own color scheme. Simply copy one of the existing color schemes, define the colors to your liking and include it into your site.

Elements

Using the Included Elements

This section describes the usage of the various elements that are included with Wise Words. For examples, see the page elements.html of the template.

Tabs

The basic HTML code for tabs looks as follows:

<div class="tabbable">
  <ul class="nav nav-tabs">
    <li class="active"><a href="#tab1" data-toggle="tab">Section 1</a></li>
    <li><a href="#tab2" data-toggle="tab">Section 2</a></li>
    <li><a href="#tab3" data-toggle="tab">Section 3</a></li>
  </ul>
  <div class="tab-content">
    <div class="tab-pane active" id="tab1">
      <p>I'm in Section 1.</p>
    </div>
    <div class="tab-pane" id="tab2">
      <p>Howdy, I'm in Section 2.</p>
    </div>
    <div class="tab-pane" id="tab3">
      <p>Howdy, I'm in Section 3.</p>
    </div>
  </div>
</div>

You may also have your tabs on the left/right/bottom of your content. For left/right tabs, simply add tabs-left or tabs-right to the <div class="tabbable"> tag:

Left:

<div class="tabbable tabs-left">

Right:

<div class="tabbable tabs-right">

To place your tabs below your content, add the class tabs-below to the <div class="tabbable"> tag AND switch the <ul> section with the <div class="tab-content"> section.

Buttons

Buttons have different syntax, depending on the usage. For a button within a form you simply use the button tag. For a button link on the other hand, apply the class button-link to a link:

<a href="#" class="button-link">Button Link</a>

You can also combine buttons with the included Glyphicons. Here are two code examples for both a button as well as a link tag:

Button:

<button class="button-icon"><i class="glyphicons-icon home white"></i></button>

Link:

<a href="#" class="button-link foricon"><i class="glyphicons-icon volume_up white"></i></a>

For more examples, refer to the page elements.html of the template. For more info on the included Glyphicons see the Icon Section of this documentation.

Tables

For tables such as the ones that appear in elements.html, use the standard HTML table syntax and add the class table, like so:

<table class="table">
    <tr>
        <th>Very Small</th>
        <th>Example Table</th>
    </tr>
    <tr>
        <td>Second Row</td>
        <td>Second Column</td>
    </tr>   
    <tr>
        <td>Third Row</td>
        <td>Second Column</td>
    </tr>       
</table>

You may add classes table-striped or table-hover to the table, to make it more visually appealing:

<table class="table table-striped table-hover">

Pagination

The syntax of a pagination box looks as follows:

<div class="pagination">
  <ul>
    <li><a href="#">«</a></li>
    <li class="active"><a href="#">1</a></li>
    <li><a href="#">2</a></li>
    <li><a href="#">»</a></li>
  </ul>
</div>

Icons

Using the Included Icon Sets

Social Icons

To place a Social Icon anywhere on your site, simply use an <i> tag with the correct class, for example:

<i class="smicon-twitter"></i>

Glyphicons

Similarly the Glyphicons are used by using the <i> tag with the correct class applied, for example:

<i class="glyphicons-icon dog"></i>

The Glyphicons are available both in black and in white. To render a white icon instead of a black on, simply add the class white to it:

<i class="glyphicons-icon dog white"></i>

This renders the icon based on the .SVG sprite included under the img folder. If you prefer to include the Glyphicons font-based, use the following sytnax:

<a href="" class="glyphicons glass"><i></i></a>
<a href="" class="glyphicons white glass"><i></i></a>

For a full list of all available icons, please see the page icons.html in the template files.

Note: You may only use the Glyphicons in together with Wise Words. If you like them and want to use them for your own product (i.e. your own template, your iPhone app or another website) you will have to buy your own license. More info
